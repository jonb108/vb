This page explains how to use the VISIBILITY BRIGADE utility.
It helps you manage your letters and messages.

It lives here: logicalpoetry.com/vb

There is a very comprehensive National Visibility Brigade Toolkit.

In the text field you enter potential messages and also various ‟commands”.

You first do a one-time initialization:
  1. Create a brigade by giving a name and optionally a password.

  2. Give the brigade a title and optionally an attachment type.

  3. Add all of your letters specifying how many you have of each.

  4. The letter boards default to black.  Change this to white if needed.


You can then test messages to see if you have enough letters.
After a message is used on the bridge you can add it to a historical list.

The letters are ALL UPPERCASE.


Brigades
Create a new Visibility Brigade:
NEW <NAME> <PASSWORD>   (no spaces in name or password)
or
NEW <NAME>          (no spaces in name - no password is wanted)

Sometimes you have two sets of letters.
Switch between them with the V command.

Switch to a Visibility Brigade:
V <NAME> <PASSWORD>   (switch to the named brigade with password)
V <NAME>                    (switch to the named brigade that has no password)

If you exit the browser the current brigade will be remembered for
when you return.


For the current brigade:

TL <TITLE>       (spaces are okay in TITLE)
    The title is displayed at the top.
    It can be a longer description than the shorter brigade name.

Letters are attached to the fence in a variety of ways.

AT <TYPE>       (AT = Attachment Type)
    TYPE can be:
        PVC             (Polyvinyl Chloride pipes)
        HOOK
        ROPE
        ZIP TIES
        etc.
    
    AT by itself will show the attachment type.

BC <COLOR>            (set letter board color)
                                   (COLOR is either BLACK or WHITE)

PW <PASSWORD>    (change Password - no spaces in password)
PW -                            (remove the Password)

RN <NEW NAME>      (rename the brigade, no spaces in new name)


Letters
BA <LETTER>  ## ...      (BA = Bulk Add)

    LETTER is a single character or one of these two letter sequences:
        SP   space
        HT   red heart ❤
        DN   down arrow ↓
        UP   up arrow ↑

    ## is a number - how many of the LETTER you have.

    You can put multiple <LETTER> ## in a row like this:

        BA  A 3  B 4  C 2  D 3  E 10  ! 1  $ 2  SP 8

    LETTER can be any character including
        punctuation (non alpha-numeric) characters.
        There is one exception: tilde ~ is not permitted.

ADD <LETTER> ...      (increment the number for the letters)

    Example: ADD  S  S  U  Z          (add two S, a U, and a Z)

SUB <LETTER>          (decrement the number for the letter
                           if there are none left the letter will be removed)

L               (List all of the letters and numbers and the total)

Compare the number of letters to those in another brigade.
CL <NAME>
CL <NAME> <PASSWORD>
                    (Using CLO will only show where there is a difference)


Testing Messages
To test a message to see if there are enough letters
enter the message in the text field and hit Return.

If there are not enough letters to make the message you will see
       which additional letters need to be made.
If there are not enough 0s or 1s it will substitute O for 0 and I for 1.

If there ARE enough letters it will say “Okay 👍 😊”
or
If there are NOT enough letters it will say “IF those are made ...”

    and then it will display:

    - how many total letter boards are needed for the message

    If the attachment type is PVC it will show:

        - how the message will actually look when supported by the PVC pipes
        - how many T, J, I, and S pipes are needed to support the message
        

        
T- pipe with a “tee”

        
J- pipe with a “union connector”

        
I- a plain pipe

        
S- a short pipe

        

You can click on the displayed message to copy it to the clipboard
so you can easily paste it.  Perhaps you want to modify it or
try the message in another brigade that has a different set of letters.


Message History
To add a message to the brigade's message history:

<DATE> <MESSAGE>

    where DATE can be:

        4/5/2025
        4/5/25
        4/5     (current year)
        4       (current month/year)
        T       (today)

    if MESSAGE is just a dash ‘-’, the message on that date will be deleted

With a successful Okay 👍 😊 message displayed on the screen
you can easily add it to the history with this:

U <DATE>        (Use this message on the date)    

Note that you CAN add more than one message on a given date.

H
    Show the complete message History in two columns: Date and Message
    with the most recent dates first.

    At the bottom is a link: CSV Export.
    Clicking on it will download a comma-separated-value formatted file
    that you can open in a spreadsheet.


Messages for ALL Brigades

LM                    (show the unique messages from ALL brigades)
LMD                  (show the message history for ALL brigades including dates)

    These two commands also have a CVS Export link at the bottom.

SM <term>       (search ALL brigade messages for the search term)


Log
LG                  (show a time stamped log of activity)

LG CLEAR     (clear the log)


Bookmarks
You can make a bookmark to go directly to your brigade.

If you have just one brigade this is the URL to bookmark:

    logicalpoetry.com/vb

If you have more than one set of letters and the brigade name is XXX 
this is the URL to bookmark:

    logicalpoetry.com/cgi-bin/vb?cmd=v+XXX

and if it has a password of YYY:

    logicalpoetry.com/cgi-bin/vb?cmd=v+XXX+YYY


IMPORTS
You can import letter counts or dated messages
from a .csv (Comma Separated Values) file.

The command IM begins the process.
You then choose a file on your computer in the proper format
and press 'Import' to import it.

A letter count file must have just two lines and look like this:
    A,B,C,F,Q,$,!
3,4,2,10,2,1,1
A dated message file can have many lines and must look like this:
    2/3/2025,POTUS IS PUTIN'S PUPPET
12/31,DEFEND DEMOCRACY
7/4/26,CELEBRATE INDEPENDENCE FROM FASCISM
The date can be in a variety of formats as shown.
